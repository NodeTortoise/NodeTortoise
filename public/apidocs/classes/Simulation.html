applyUpdate

(
  • agentStreamController
  • modelUpdate
)
Object

Definido en public\js\Simulation.js:74

Este método sobre-escribe el método AgentStreamController.prototype.applyUpdate del código original de Tortoise. Se encarga de controlar la actualización de la simulación (a nivel gráfico) y de los valores de los outputs. En el caso del cliente maestro, se envía un mensaje al servidor para que los clientes realicen la actualización y finalmente se ejecuta el código original de actualización. En el caso los clientes que no son maestros, se ignora cualquier proceso. En caso de que aún no se hara realizado la conexión con el socket, se ejecuta el código original de actualización, en ambos casos.

Parameters:

  • agentStreamController Object

    El objeto AgentStreamController del modelo

  • modelUpdate Object

    El objeto que contiene las intrucciones de actualización

Returns:

Object:

El resultado del proceso de actualiación. Este resultado proviene de la lógica del código de Tortoise
