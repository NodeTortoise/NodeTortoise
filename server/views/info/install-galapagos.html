Instalación utilizando Script

A continuación se describe los pasos de instalación de Galapagos utilizando el script de instalación. Los comandos mostrados son para cualquier sistema operativo Linux, no obstante, no existen scripts para todos ellos.

  1. Descargar el script de instalación adecuado para su sistema operativo, desde el repositorio de GitHub:
    Ubuntu: Descargar | Ver en GitHub
  2. Subir el archivo al servidor y acceder a la ubicación de éste.
  3. Dar permisos de ejecución al archivo chmod 771 Install-Galapagos.sh
  4. Ejecutar el archivo con permisos de root 
    sudo ./Install-Galapagos.sh
  5. Seguir los pasos que se indiquen, aceptando la instalación de los paquetes necesarios.

Instalación Manual

A continuación se describe los pasos de instalación de Galapagos. Los comandos mostrados son para el sistema operativo Ubuntu. No obstante, los pasos a realizar son los mismos para otros sistemas operativos.

  1. Instalar NodeJS 
    apt-get install nodejs-legacy
  2. Instalar NodeJS Package Manager (NPM) 
    apt-get install npm
  3. Instalar SBT 
    echo "deb https://dl.bintray.com/sbt/debian /" | tee -a /etc/apt/sources.list.d/sbt.list
apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv 642AC823
apt-get update
apt-get install sbt
  4. Instalar GIT 
    apt-get install git
  5. Instalar UnZip 
    apt-get install unzip
  6. Desinstalar OpenJDK. Se debe realizar este paso, porque se necesitar contar con la versión 8 del JDK, pero a la fecha (2015-09-19) no estaba aún disponible. 
    apt-get purge $OPENJDK_PKG*
  7. Instalar Oracle Java 8 JDK 
    add-apt-repository ppa:webupd8team/java
 apt-get update
 apt-get install openjdk
  8. Instalar Play Framework 
    wget http://downloads.typesafe.com/typesafe-activator/1.3.5/typesafe-activator-1.3.5-minimal.zip
 unzip typesafe-activator-1.3.5-minimal.zip
 rm typesafe-activator-1.3.5-minimal.zip
  9. Obtener repositorio de Galapagos en GitHub 
    git clone https://github.com/oscarmartinezm/Galapagos.git
  10. Ingresar al direction donde está ubicado Galagos 
    cd Galapagos
  11. Crear link simbólico a Play Framework dentro de Galapagos 
    ln -s ../activator-1.3.5-minimal/activator start
  12. Instalar UnZip 
    apt-get install unzip

Ejecución

A continuación se describe los pasos para ejecutar Galapagos. Los comandos mostrados son para cualquier sistema operativo Linux.

  1. Ingresar al directorio donde está ubicado Galapagos 
    cd Galapagos
  2. Ejecutar el archivo start, con derechos de root 
    sudo ./start run
