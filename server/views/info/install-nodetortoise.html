Instalación utilizando Script

A continuación se describe los pasos de instalación de NodeTortoise utilizando el script de instalación. Los comandos mostrados son para cualquier sistema operativo Linux, no obstante, no existen scripts para todos ellos.

  1. Descargar el script de instalación adecuado para su sistema operativo, desde el repositorio de GitHub:
    Ubuntu: Descargar | Ver en GitHub
  2. Subir el archivo al servidor y acceder a la ubicación de éste.
  3. Ejecutar el archivo con permisos de root 
    sudo ./Install-NodeTortoise.sh
  4. Seguir los pasos que se indiquen, aceptando la instalación de los paquetes necesarios.

Instalación Manual

A continuación se describe los pasos de instalación de NodeTortoise. Los comandos mostrados son para el sistema operativo Ubuntu. No obstante, los pasos a realizar son los mismos para otros sistemas operativos.

  1. Instalar NodeJS 
    apt-get install nodejs-legacy
  2. Instalar NodeJS Package Manager (NPM) 
    apt-get install npm
  3. Instalar GIT 
    apt-get install git
  4. Instalar NodeJS Forever 
    npm install -g forever
  5. Obtener repositorio de NodeTortoise en GitHub 
    git clone https://github.com/oscarmartinezm/NodeTortoise.git
  6. Ingresar al direction donde está ubicado NodeTortoise 
    cd NodeTortoise
  7. Instalar las extensiones de NodeJS necesarias para la ejecución de la aplicación 
    npm install

Ejecución vía script

A continuación se describe los pasos para ejecutar NodeTortoise utilizando el script. Los comandos mostrados son para cualquier sistema operativo Linux.

  1. Ingresar al directorio donde está ubicado NodeTortoise 
    cd NodeTortoise
  2. Elevar como usuario root (necesario para ejecutar en puerto 80) 
    sudo su
  3. Ejecutar el archivo run 
    . ./run

Ejecución manual

A continuación se describe los pasos para ejecutar NodeTortoise manualmente. Los comandos mostrados son para cualquier sistema operativo Linux.

  1. Ingresar al directorio donde está ubicado NodeTortoise 
    cd NodeTortoise
  2. Establecer las variables de entorno necesarias 
    export PORT="80"
export NODE_ENV="production"
  3. Ejecutar la aplicación 
    forever start ./bin/www
